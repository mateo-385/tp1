# Proyecto React Native - TP1

## Instrucciones

1. Navega al directorio `server`:

   ```bash
   cd server
   ```

2. Instala las dependencias:

   ```bash
   npm install
   ```

3. Abre el archivo HTML con Live Server para visualizar el proyecto.

## Requisitos

- Node.js instalado.
- Extensión Live Server en tu editor de código.
